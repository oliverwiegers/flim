# Flim

> Neovim flake kept simple intentionally.

## Features

- Code completion and Snippets
- Syntax highlighting
- Linting

### Visuals

- Gruvbox Theme
- Lualine statusline and tabline
- Treesitter
- Fancy icons
- Highlight spaces and tabs

### Language Support

- Lua
- Nix
- Python
- Markdown
